# Debt Payoff Calculator

**Compare Snowball vs Avalanche strategies** — Find the fastest way to become debt-free.

## Live Demo

Deployed on Netlify (add your link after deployment)

## Features

- **Snowball vs Avalanche** comparison with real simulation
- Dynamic debt reduction chart
- Fully bilingual (English + 繁體中文)
- Dark / Light mode
- Auto-save with localStorage
- PDF report export
- AdSense ready slots
- Mobile responsive

## How to Deploy to Netlify (Super Easy)

1. Go to [Netlify](https://app.netlify.com)
2. Click **"Add new site"** → **"Import an existing project"**
3. Connect your GitHub account
4. Select the `debt-payoff-calculator` repository
5. Click **Deploy site**
6. Done! Your calculator is live in ~1 minute

## Tech Stack

- Vanilla HTML + Tailwind CSS
- Chart.js
- jsPDF
- Fully client-side (no backend needed)

## Customization

- Replace the AdSense placeholder with your real ad code
- Change default sample debts in the code
- Modify colors in Tailwind classes
